Polytechnic of Turin - Master Degree

2026

This thesis explores the application of machine learning techniques to log analysis for enhancing software reliability. The research demonstrates how ML-driven approaches can enable proactive detection of system anomalies, predict potential failures, and provide actionable insights for software maintenance. Through practical implementation and case study, the work showcases the effectiveness of these techniques in production environments using tools like Splunk MLTK.
